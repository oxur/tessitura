# Tessitura Design Documentation

This directory contains the design documentation for the Tessitura project:

- **docs/** - Oxur Design Documents (ODDs), organized by state
- **dev/** - Development notes and planning documents

## Managing Documentation

The documentation is managed using the `odm` CLI tool from the `oxur-odm` crate.

See `odm --help` for CLI usage.

## Document Index

See [docs/index.md](docs/index.md) for the full list of design documents.
